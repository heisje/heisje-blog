## 기술 스택
- Next.js
- TypeScript
- Contentlayer
- tailwind
- Docker
- Jenkins

## 환경 설정
1. pnpm 패키지 설치
```
pnpm install
```
2. 개발서버 시작
```
next dev
```
3. http://localhost:3000/
에서 확인합니다.
