## 20. (웹 : 영화 추천 서비스)

- 기간 : 2022.11 (4주)
- 개요 : 20글자의 짧은 리뷰를 통해 데이터를 쌓고, 영화를 추천받는 웹 서비스
- 성과 : 🏆 1학기 프로젝트 경진대회 우수팀 수상 (삼성전자)
- 팀 내 역할 (2인, 풀스택)

  - 영화 데이터 수집
  - ERD 설계
  - LazyLoading과 Pagination을 사용해 효율적인 영화 리스트 구현
  - 유사 사용자와 잠재적 점수를 활용한 사용자 추천 알고리즘 설계
