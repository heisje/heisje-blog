## 토니버니 (웹 : 통역 화상 서비스)

### 📝 개요

- 실시간/예약 통역 번역기능을 제공하는 웹 플랫폼
- 팀원 : 6 FullStack
- 기간 : 2023.01.03 ~ 2023.02.17 (7주)

### 📝 성과

- 🏆 [2학기] 공통 프로젝트 우수팀 수상 (8팀 중 3등) (삼성전자)

### 📝 기술 스택

`Vue`, `Vuex`, `SCSS`, `JavaScript`, `Spring Boot`, `JPA`

### 📝 내 역할 요약

- 프론트엔드:
  - UI구현 (헬퍼 및 고객 유저 기능 / 포트폴리오 / 즉시 통역 / 예약 통역 / 번역 / 통역 프로세스 / 번역 프로세스 / 공고 / 리뷰 / 신고 / 좋아요 / 공지 / 약관 / 커뮤니티 등)
  - 공통 컴포넌트 (모달, 토스트, 버튼, Input, Tag, Card 등 디자인 시스템 기반)
  - 페이지 라우터 설계 및 제작
  - JWT 미들웨어 구축
- 백엔드: 커뮤니티 API 구현
  - `Spring Boot`, `JPA`를 사용해 커뮤니티 서비스 로직 구현

## 회고

- 컨벤션, API연동 규격서, ERD, 간트차트, 시퀀스다이어그램 등 문서화를 통해 명확한 개발이 가능했습니다.
- 공통 컴포넌트의 재사용으로 63개의 페이지를 빠른 시간 내에 제작할 수 있었습니다.
- 공통 컴포넌트의 설명을 주석으로만 달아서 이해가 어려웠습니다. Storybook과 같은 라이브러리의 필요성이 재고되었습니다.
- 코드리뷰를 진행하며 많이 배우고 성장할 수 있는 기회가 되었습니다.
- 전원 풀스택이였지만 대 기능별로 담당자를 제안하여 명확한 업무분담이 잘 이루어졌습니다.
- 백엔드와 프론트엔드 전반을 겪어보며 빈번한 API의 수정이 일어났다. API 설계의 중요성과 필요성을 느꼈습니다.
- 사용자의 분리(번역가/고객)양측의 니즈를 충족시키는 것이 어려웠습니다.
